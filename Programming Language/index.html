Programming Languages

James Gosling

*Java is an extremely transferable programming language used across platforms and different types of devices, from smartphones to smart TVs.

History:
  • 1991: Java project started (called Oak) by James Gosling at Sun Microsystems.
  • 1995: Java 1.0 released – “Write Once, Run Anywhere”.
  • 2023: Java 21 (LTS) – Virtual threads (Project Loom).

Guido van Rossum

*Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

History:
  • 1989: Python started by Guido van Rossum in the Netherlands.
  • 1991: First release of Python 0.9.0 – Included functions, exceptions, etc.
  • 2024: Python 3.12 – Faster, better error messages.

Bjarne Stroustrup

*C++ is one of the world's most popular programming languages. C++ can be found in today's operating systems, Graphical User Interfaces, and embedded systems.

History:
  • 1979: Started by Bjarne Stroustrup at Bell Labs (called "C with Classes").
  • 1985: First release of C++ (official name).
  • 2023: C++23 – More improvements (consteval, standard library updates).
